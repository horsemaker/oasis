## Tech Stack

**Client:** React, React Router, Context API, Horsemaker's UI

**Server:** Mockbee (Mock Backend)

## Run Locally

Clone the project

```bash
  git clone https://github.com/horsemaker/oasis.git
```

Go to the project directory

```bash
  cd oasis
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
